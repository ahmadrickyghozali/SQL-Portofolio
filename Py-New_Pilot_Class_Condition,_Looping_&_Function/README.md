### 1. Conditional Logic (Percabangan)

### 2. Looping & Iteration (Perulangan)
- Teknik untuk menjalankan blok kode secara berulang untuk efisiensi.

- Kontrol perulangan dengan `break`, `continue`, dan `pass`.

- Mempelajari cara menulis kode yang reusable (dapat digunakan kembali) dan rapi.
